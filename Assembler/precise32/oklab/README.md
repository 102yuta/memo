# oklabのGAS入門を読み解く

## 8080(8bitCPU)のイメージ

```
---- CPU ----------------------------
|                      8       1    |
| [ ALU      ] --- [ A:0000 0000]   |
|                  [ B:0000 0000]   |
|                  [ C:0000 0000]   |
|                  [ D:0000 0000]   |
|                                   |
-------------------------------------
```

## 8086(16bitCPU)のイメージ

8bitのレジスタを拡張(eXtend)してAX, BX, CX, DXという名前になった。

```
      8086のイメージの一部
      ---- CPU ---------------------------------------
      |                      16         8       1    |
      | [ ALU      ] --- [ AX:0000 0000 0000 0000]   |
      |                  [ BX:0000 0000 0000 0000]   |
      |                  [ CX:0000 0000 0000 0000]   |
      |                  [ DX:0000 0000 0000 0000]   |
      |                                              |
      ------------------------------------------------
```
